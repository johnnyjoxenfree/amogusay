 # amogusay
 cowsay, but for among us

 ## Installation
 ```bash
 git clone https://github.com/johnnyjoxenfree/amogusay.git
 ```
 or download the `.zip` file, go to the download directory then run `install.sh`
 ```bash
 bash install.sh
 ```
